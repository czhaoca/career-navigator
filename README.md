## Mission
To build a comprehensive AI-powered platform that empowers professionals to discover their career potential, craft compelling professional narratives, and navigate their career journey with confidence and strategic insight.

## Overview
Career Navigator is an intelligent career development platform that combines AI-powered guidance with sophisticated experience management to help professionals at every stage of their journey - from career exploration and mentorship to resume building and professional networking.

## Technology Architecture

### Privacy-First Design
- **User Data Sovereignty**: Complete control over data location and access
- **Zero-Knowledge Architecture**: No administrative access to encrypted user data
- **BYOK Implementation**: Bring Your Own Keys for LLM API access
- **End-to-End Encryption**: Military-grade security for all data layers

### AI Integration Options
- **API-Based**: Support for OpenAI, Anthropic, and enterprise AI services
- **Self-Hosted**: Local LLM deployment with Ollama and similar platforms
- **Subscription Models**: Integration with ChatGPT Plus and Claude Pro
- **MCP Integration**: Model Context Protocol for contextual conversations

### Database Flexibility
- **Oracle Cloud Free Tier**: Always-free ATP database with enterprise security
- **Cloudflare D1**: Global edge database with built-in replication
- **Multi-Cloud Support**: AWS, Azure, and Google Cloud compatibility
- **User-Controlled**: Database hosted in user's own cloud account

## Getting Started

### For Individual Users
1. **Set Up Your Profile**: Import experiences and configure privacy settings
2. **Explore Career Paths**: Use AI-powered discovery tools to identify opportunities
3. **Develop Your Stories**: Transform experiences into compelling professional narratives
4. **Build Your Resume**: Generate optimized resumes for target opportunities
5. **Plan Your Network**: Prepare for strategic professional relationship building

### For Developers
1. **Clone Repository**: `git clone https://github.com/czhaoca/career-navigator.git`
2. **Install Dependencies**: `npm install`
3. **Configure Environment**: Set up database and AI API connections
4. **Run Development Server**: `npm run dev`
5. **Explore Add-ons**: Review specialized modules in `addons/` directory

### For Organizations
1. **Enterprise Deployment**: White-label solutions for educational institutions
2. **Custom Integrations**: API access for existing HR and career services platforms
3. **Training Programs**: Professional development curriculum integration
4. **Analytics Dashboard**: Aggregate insights while maintaining individual privacy
